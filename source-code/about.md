# Achievements Plus

**Achievements Plus** introduces unique achievements to Geometry Dash, expanding gameplay with new challenges, milestones, and progression for dedicated players.

If you experience any problems, bugs, or unexpected behavior while using this mod, feel free to contact me or report the issue on the GitHub page. Feedback and reports are always appreciated and help improve the mod over time.

Feel free to suggest new achievements for the mod!
